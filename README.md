---

## 🌎 About

**Sabores de Latinoamérica** is an informational website exploring the food culture of Latin American nations, with a dedicated page for each country's traditional cuisine, ingredients, and dishes.

### Countries covered:
- 🇵🇦 Panama
- 🇨🇴 Colombia
- 🇨🇷 Costa Rica
- 🇸🇻 El Salvador
- 🇭🇳 Honduras

---

## 🚀 How to Run

No build step required — just open the file directly:

```bash
git clone https://github.com/deadlyrat/sabores.git
cd sabores
# Open index.html in your browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

## 🎓 Context

Built as a **university project** for a web fundamentals course at Universidad Tecnológica de Panamá. Demonstrates HTML structure, CSS styling, and multi-page static site organization.
